# JCC Portfolio

## Requirements

- NodeJS 16
- VueJS CLI
- `pnpm`

### For VSCode Dev Environment

- Docker
- Remote Extension Pack for VSCode

## Basic Setup

### Normal

- Run required packages `pnpm install -g eslint prettier`
- Then run `pnpm install` 

### VSCode

- Run VSCode then open the project directory
- Then reopen in container

## Building

- `pnpm run build`

## Testing

- `pnpm run serve`
