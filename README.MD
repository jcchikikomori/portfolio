## Requirements

- NodeJS 18
- VueJS CLI
- `pnpm`

## Setup

### First step

- Run required packages `pnpm install -g eslint prettier`
- Then run `pnpm install` 

### Building

- `pnpm run build`

## Testing

- `pnpm run serve`
