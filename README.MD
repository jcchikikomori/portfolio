# JCC Portfolio

## Requirements
- NodeJS
- Yarn
- Ruby (optional)

## Basic Setup
- Run `yarn install` 
- Optional: If you have ruby, run the following: 
	- `cd && gem install --no-ri --no-rdoc bundler`
	- `bundle install`
	- Then test the following if exist(s): 
		- `which bundler`
		- `which sass`

## Required TODOs
- For HTML tests (Homepage): Run `gulp homepage`
- To generate CSS from SASS: Run `gulp csass`
