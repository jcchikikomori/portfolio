# portfolio

Personal portfolio site built with Vue 3, Vite, and Bootstrap 5.

## Requirements

- Node.js 22 LTS (see `.nvmrc`)
- pnpm

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm run dev
```

## Build

```bash
pnpm run build
```

## Preview production build

```bash
pnpm run preview
```

## Testing

```bash
pnpm run test
```

## Linting and formatting

```bash
pnpm run lint
pnpm run format
```
